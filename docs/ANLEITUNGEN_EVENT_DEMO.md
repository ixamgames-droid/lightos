## Workflow-Anleitung (VC selbst bearbeiten)

- **[➜ Effekte einfach aufbauen (Smart-Build)](anleitung_vc_smartbuild/ANLEITUNG.md)** —
  der **neue, anfängerfreundliche Aufbau-Flow**: Effekt reinziehen → in der
  **„Effekt einrichten"-Karte** ankreuzen, was er können soll → fertig (ein Strg+Z macht
  alles rückgängig). Plus **grafische Widget-Galerie**, **Schutz vor Doppelbelegung**
  (Ersetzen / Dazu koppeln / Neues Widget) und **Highlighting** zusammengehöriger Elemente.
- **[➜ Virtual Console live bearbeiten (klassischer/manueller Ablauf)](anleitung_vc_workflow/ANLEITUNG_VC_WORKFLOW.md)** —
  Elemente löschen, hinzufügen, einen **Effekt an einen An-/Aus-Schalter binden**,
  **Speed-/Submaster-/Tempo-Bus-Fader** anlegen und **speichern**. (Komplett live durchgeklickt.)
- **[➜ Alle VC-Elemente (Referenz)](anleitung_vc_elemente/ANLEITUNG_VC_ELEMENTE.md)** —
  alle 15 Bau-Elemente (was/wie bedienen/was einstellbar), die Baukasten-Knöpfe und die
  neue **Chase-auf-Gruppe**-Auswahl.
